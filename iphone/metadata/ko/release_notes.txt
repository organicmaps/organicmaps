• EU 사용자는 이제 iOS 설정 → 앱 → 기본 앱 → 내비게이션에서 Organic Maps를 기본 내비게이션 앱으로 설정할 수 있습니다

• 2025년 10월 21일자 최신 OpenStreetMap 데이터
• 새로운 지도 객체: 모니터링 스테이션, 교통 섬, 크나이프 수치료 시설, 미니어처 철도, 야외 스타일 캠핑장, 공항 터미널, 실내 놀이 공간, 통신 매장, 보트 대여 시설 및 보트 진수 시설
• 지하철 입구 이름이 이제 지도에 표시됩니다
• 보행자 내비게이션 모드에서 걸을 때 위치 화살표가 이동 방향을 표시합니다
• 내비게이션용 음성 안내를 포함한 슬로베니아어 지원 추가
• 폐기물 처리 및 매립지 아이콘 업데이트
• 내비게이션 모드에서 흰색 배경의 흰색 상태 표시줄 수정
• 내비게이션 시작 버튼 크기 증가
• iPad에서 경로 계획 시 빈 공백 제거
• 하이킹 트레일에 대한 추가 색상 지원
• 강 또는 수로 세그먼트 선택 시 표시되는 정보 개선
• 모든 언어에서 개선된 동의어를 사용한 전기차 충전소 검색 개선
• 이모지 검색 수정 (네, 검색창에 이모지를 입력할 수 있습니다!)
• 정확한 주소 검색에 대해 너무 많은 결과를 표시하던 검색 힌트 수정
• https://umap.openstreetmap.fr/ 에서 GeoJSON을 가져오고 내보내기할 때 이제 모든 메타데이터를 보존합니다

### 놓쳤을 수 있는 최근 출시된 기능

• 선택한 트랙 포인트에 북마크 저장
• 먼저 저장하지 않고 녹음 트랙 삭제
• GeoJSON 파일 가져오기
