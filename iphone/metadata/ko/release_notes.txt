- 대중교통 정류장을 탭하면 대중교통 지도 스타일에서와 같이 전체 경로가 지도에 강조 표시됩니다.
- 장거리 대중교통 노선과 다른 OpenStreetMap 관계가 이제 지도 경계를 넘어 하나의 연속된 선으로 연결됩니다.
- 5월 3일자로 업데이트된 OpenStreetMap 데이터입니다.
- 베트남과 말레이시아는 이제 더 작은 지역으로 나뉘어 별도로 다운로드할 수 있습니다.
- 홍콩, 마카오, 하이난은 이제 광둥성에서 분리되어 인접한 경계선이 업데이트되었습니다.
- 인도네시아, 말레이시아, 탄자니아, 태국, 베트남의 등고선을 업데이트했습니다.
- 내비게이션 정보 패널에 트랙 녹화 버튼을 추가했습니다.
- 활화산, 수로 접근 지점, 슬립웨이, 물담배 라운지, 건설 중인 건물 등 새로운 OpenStreetMap 유형을 추가했습니다.
- 뷰포인트의 검색 동의어로 '전망대'를 추가하고 미국 도로 동의어로 'pkwy'를 추가했습니다.
- 가독성을 높이기 위해 보행자 구역의 색상을 약간 더 어둡게 변경했습니다.
- 지도의 북마크 레이블이 더 이상 겹치지 않습니다.
- 이제 여러 줄의 텍스트가 깔끔하게 잘리고 도로명에 더 이상 중복이 포함되지 않습니다.
- 경로 설정: 이제 경로를 재개하면 이미 통과한 중간 지점이 제거됩니다.
- TestFlight 빌드에서 잘못된 북마크 목록이 사용되던 문제를 수정했습니다.
- KMB 파일 가져오기를 수정했습니다.
- 카테고리, 검색 동의어, UI 문자열의 번역을 교정했습니다. 뉴사우스웨일즈 및 노던 테리토리 설명에 대한 번역을 추가했습니다.
- 글꼴 모양, 글리프 캐싱, 경로 선 클리핑 등 안정성과 성능이 개선되었습니다.
- 트랙과 경로의 고도 차트를 개선했습니다.
- TTS, 설정, 트랙 녹음 등의 작업을 위해 내비게이션 메뉴 버튼을 더 크게 만들었습니다.
- 장소 페이지에서 확장 가능한 설명 섹션의 애니메이션을 개선했습니다.
- 고도 차트 스크러빙이 종료되면 장소 정보 페이지가 더 이상 닫히지 않고, 예기치 않게 튕기지 않으며, 키보드를 열어 편집하는 동안 제목이 계속 표시됩니다.
- 더 이상 존재하지 않는 트랙이나 북마크를 삭제할 때 발생하는 크래시를 수정했습니다.
