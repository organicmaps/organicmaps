- ใหม่! แตะเพื่อเลือกแสดงเส้นทางหรือเส้นทางซ้อนทับ
- ใหม่! เลือกสีใดก็ได้สำหรับที่คั่นหนังสือ
- ใหม่! รองรับพิกัด British National Grid (OS Grid), Irish Grid และ Irish Transverse Mercator (ITM)
- ใหม่! รองรับการเข้าถึงสำหรับ Dynamic Type และฟอนต์ขนาดใหญ่
- ทดลองใช้: เปิดใช้งานภาพถ่ายดาวเทียมในการตั้งค่า Organic Maps โดยใช้ URL ของเซิร์ฟเวอร์ไทล์แรสเตอร์ที่กำหนดเอง
- การจัดรูปแบบตารางที่สะอาดขึ้นใน UI
- พิมพ์ `?map-download-server:https://your-server.com/` ในหน้าต่างค้นหาเพื่อแทนที่เซิร์ฟเวอร์ดาวน์โหลดแผนที่ Organic Maps พิมพ์ `?no-map-download-server` เพื่อลบการแทนที่
- การแก้ไขข้อบกพร่องต่าง ๆ และการปรับปรุงประสิทธิภาพ

## ข้อมูลแผนที่

- ข้อมูล OpenStreetMap ได้รับการอัปเดต ณ วันที่ 24 มิถุนายน
- ข้อมูล Wikipedia อัปเดตถึงวันที่ 20 มิถุนายน รวมถึงบทความภาษาอิตาลี

## การแสดงแผนที่และรูปแบบ

- รูปแบบที่ดูสวยงามยิ่งขึ้นสำหรับพื้นที่ชายหาด ทราย หินกรวด ภูเขาหินโล้น สวนผลไม้ และไร่องุ่น รวมถึงการฟักไข่ในพื้นที่คุ้มครองและพื้นที่ชุ่มน้ำ
- ป้ายข้อความโค้งที่เรียบเนียนยิ่งขึ้นสำหรับถนนและแม่น้ำ
- แม่น้ำ ลำธาร และพื้นที่ชุ่มน้ำมีความคมชัดสูงกว่าและมองเห็นได้ชัดเจนยิ่งขึ้นในโหมดมืด
- ไอคอนใหม่และปรับปรุงสำหรับการค้นหาหมวดหมู่และผลการค้นหาบนแผนที่
- ปรับปรุงการจัดรูปแบบและการแสดงผลข้อความหลายภาษา

## เส้นทาง, บุ๊กมาร์ก & เส้นทาง

- ปรับปรุงการแสดงผล HTML ของคำอธิบายบุ๊กมาร์กและแทร็ก
- GeoJSON ติดตามระดับความสูงและเวลาขณะนำเข้าและส่งออก
- แก้ไขปัญหาการหยุดทำงานหลังจากย้ายแทร็กไปยังรายการอื่น
- ขณะนี้มีการแสดงเว็บไซต์สำหรับแหล่งมรดกทางวัฒนธรรม
- ชื่อของผู้ดำเนินการจะแสดงสำหรับผลการค้นหาที่ไม่มีชื่อแล้ว ตัวอย่างเช่น ATM ที่ไม่มีชื่อจะแสดงชื่อธนาคารของมัน
- บรรณาธิการ: แก้ไขข้อความคำอธิบายสำหรับการแก้ไข/ชุดการเปลี่ยนแปลงอาคารอพาร์ตเมนต์ OpenStreetMap
- ปรับปรุงการตรวจจับ HTML ในคำอธิบายบุ๊กมาร์กและแทร็ก
- เส้นทางที่ซ้ำกันถูกกรองออกแล้ว

## การแปล

- คำแนะนำเสียงแบบทีละขั้นตอนเป็นภาษากวางตุ้ง
- อัปเดตการแปลภาษาเยอรมันและฝรั่งเศส
- แก้ไขการแปลเสียงนำทาง "onto street" ที่ไม่ถูกต้องสำหรับภาษาจีน, ภาษาเซอร์เบีย และภาษาคาตาลัน
