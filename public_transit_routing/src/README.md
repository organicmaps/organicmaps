# routing_service

## Overview
This repository provides a routing service built on OpenStreetMap (OSM) data.  
It includes tools to generate routing indices, a Go‑based HTTP API, and a Python test suite that validates the API.

**⚠️ Experimental Note:** This project is an experimental prototype. It is not yet integrated with Organic Maps and may be moved to a dedicated GitHub repository in the future. Please treat it as a proof‑of‑concept rather than production‑grade code.
