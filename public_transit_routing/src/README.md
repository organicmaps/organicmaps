# public_transit_routing/src/README.md

## Overview
`public_transit_routing` provides a lightweight HTTP service that calculates public‑transport routes using pre‑computed routing indices.  
The repository ships with a **src** folder that contains the service implementation, data‑processing scripts, and test suite.

> **Why the folder name?**  
> The original folder name (`job_github_bounty_organicmaps_organicmaps#5331`) was generated by an automated job. It has been renamed to a conventional, easy‑to‑understand layout (`public_transit_routing/src`).
